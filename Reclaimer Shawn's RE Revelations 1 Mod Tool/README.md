# Known Glitches
* You can kill yourself with your explosive weapons while using Insta-Kill, even if Infinite Ammo is enabled too. Use weapons like the knife or non-explosives to get around this. I could fix this but won't.
* As of right now, the Item Editor will allow you to select Jessica's G18, Jill's Samurai Edge, and Parker's Government. However, they appear in-game as question marks and are unusable. I'll try to fix this.
