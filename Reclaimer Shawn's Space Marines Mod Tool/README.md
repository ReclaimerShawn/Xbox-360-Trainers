# Known Glitches
- Stronger enemies that have shields, like Chaos Marines, also get infinite life. You can still kill them buy stunning them and clicking B. If you for some reason cannot stun the enemy given your distance, just disable the cheat and kill them, then re-enable the cheat.
- The cheats may get spontaneously turned off, especially after missions have been completed. If they do, uncheck and recheck the cheat check box and the cheats will start working again. So far, this has only occurred once with Infinite Life in my testing.
